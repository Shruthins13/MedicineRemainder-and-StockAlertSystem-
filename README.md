Medicine Reminder & Stock Alert System
📌 Project Overview

The Medicine Reminder & Stock Alert System is a desktop-based application developed using Java Swing, JDBC, and MySQL.
It helps users manage medicine inventory efficiently by tracking stock levels, expiry dates, and supplier details.

This system is especially useful for:

Pharmacies 🏥
Hospitals 🏨
Personal medicine tracking 💊


🚀 Features
➕ Add new medicines with supplier details
📋 View all medicines in a table format
⚠️ Low stock alert (when quantity ≤ 5)
📅 Expiry date tracking
🔄 Auto-refresh table after updates
🖥️ User-friendly GUI using Swing


🛠️ Technologies Used
Java (Core + Swing for GUI)
JDBC (Database connectivity)
MySQL (Database)
IDE:  Eclipse 

📸 System Modules
1. Add Medicine
Enter name, quantity, expiry date, and supplier
Saves data into MySQL database
2. View Medicines
Displays all records in a JTable
Automatically refreshes after insertion
3. Stock Alert
Shows medicines with low quantity (≤ 5)


🔥 Key Highlights
Clean MVC-like structure (Model, DAO, UI separation)
Real-time database interaction using JDBC
Simple and beginner-friendly implementation
Easily extendable for advanced features
